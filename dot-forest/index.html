입력 방식 안내

  • 키보드: 방향키 또는 WASD
  • 모바일: 화면 D-pad 터치
  • 음성: “앞으로”, “뒤로”, “왼쪽”, “오른쪽”, “먹기”
  • DotPad 패닝키: 방향 입력과 동일한 movePlayer(direction) 구조에 연결 가능
  • 정보 듣기: 1·2·3·4 키 또는 화면 버튼 — 위치 · 주변 · 미션 · 촉각 지도 (DotPad는 F1~F4)
