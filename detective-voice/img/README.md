# Detective Voice — 이미지 폴더

Nano Banana로 만든 픽셀 아트 이미지 14장을 **정확한 파일명**으로 이 폴더에 저장.

## 파일명 (이대로 정확하게)

### 장면 4장 (4:3 비율)
- `splash.png` — CRT TV with sound waveform (타이틀)
- `wine_bar.png` — 와인바 루나 사건 현장 (POLICE LINE)
- `alley.png` — 비 내리는 골목 (24H 편의점, 가로등)
- `morgue.png` — 부검실 (검안대, 시트, 의료 도구)

### 단서 6장 (5:3 비율)
- `ev_broken_bottle.png` — 깨진 와인병 (EVIDENCE 01)
- `ev_receipt.png` — POS 영수증 (EVIDENCE 02)
- `ev_text.png` — 휴대폰 메시지 (EVIDENCE 03)
- `ev_inventory.png` — 와인 재고 장부 (EVIDENCE 04)
- `ev_cctv.png` — 편의점 CCTV (EVIDENCE 05)
- `ev_autopsy.png` — 부검 보고서 (EVIDENCE 06)

### NPC 포트레이트 4장 (1:1 비율)
- `npc_park.png` — 박준영 (45세, 정장, 가는 눈)
- `npc_kim.png` — 김유진 (42세, 긴 머리, 눈물)
- `npc_lee.png` — 이민지 (29세, 포니테일, 앞치마)
- `npc_choi.png` — 최동수 (55세, B 모자, 콧수염)

## 저장 방법 (Mac)

1. 채팅에서 각 이미지 우클릭 → "이미지를 다음 이름으로 저장..."
2. 파일명 입력란에 위 정확한 이름 입력 (예: `splash.png`)
3. 저장 위치: 이 `img/` 폴더
4. 14장 다 저장한 후 `demo_v6.html`을 더블클릭하면 즉시 적용됨
